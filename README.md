# Exercício de POO

Uma API desenvolvida para gerenciar videos, com o objetivo de seguir o paradigma da POO.

## Essa API de E-commerce contém as seguintes funcionalidades:

- Video:
    - Consultar a listagem total
    - Cadastrar
    - Deletar
    - Editar os dados
